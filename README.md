# 📚 BOOKS of RUHA — Premium Aesthetic Literary Boutique

A responsive, modern, and highly interactive e-Commerce bookstore web application built for the **ikftech Web Development Internship Assessment**. This project showcases handcrafted UI/UX design, modular frontend architectures, dynamic state systems, and performance optimizations.

🔗 **Live Link:** [https://rumaisashaikhi.github.io/ikftech-project/](https://rumaisashaikhi.github.io/ikftech-project/)  
📂 **GitHub Repository:** [https://github.com/rumaisashaikhi/ikftech-project](https://github.com/rumaisashaikhi/ikftech-project)

## ✨ Features Implemented

### 🛠️ Core Requirements (100% Completed)
* **5 Required Pages:** Fully integrated production-ready pages including **Home**, **Shop**, **Product Details**, **About**, and **Contact**.
* **Responsive Engine:** Implemented mobile-first layouts with smooth breakpoints across mobile viewports, tablets, and wide desktop grids.
* **Aesthetic UI/UX:** Clean typographic hierarchy using a tailored color palette featuring soft fuchsia, deep purples, and indigo gradients.
* **Live Product Search:** Real-time multi-criteria client-side search indexing instantly processing book titles, authors, and categories.
* **Functional Contact UI:** Validation-ready elegant contact wrapper with an intuitive interactive structure.

### 🚀 Bonus Highlights (Framework Level Enhancements)
* **Dynamic Client-Side Wishlist Engine:** Users can seamlessly toggle items to a localized state (`localStorage`), instantly saving state variables asynchronously.
* **Smart Product Filtering:** Category-based filters operating concurrently with real-time UI card modifications.
* **Aesthetic Dark Toggle:** Manual override switcher embedded securely inside the Home navigation bar for an instant layout mood shift.
* **Micro-Animations & Smooth Layout Transitions:** Integrated CSS layout load states, interactive scaling effects on button hovers, and hardware-accelerated translation layers.
* **SEO Optimization Elements:** Built with high accessibility practices, structural metadata semantic tags, explicitly managed target graphs, and preload hints.

---

## 🎨 Technology Stack
* **HTML5:** Fully semantic structuring (`<header>`, `<main>`, `<nav>`, `<footer >`).
* **Tailwind CSS v4 (Browser Engine):** Modern utility-first layout styling architecture utilizing modern native properties.
* **JavaScript (ES6+ Native Engine):** Dynamic DOM modification vectors, state processing cycles, URL parsing engines (`URLSearchParams`), and client-side data streaming.
